GChartWrapper - Google Chart API Wrapper

The wrapper can render the URL of the Google chart based on your parameters. With the chart you can render an HTML img tag to insert into webpages on the fly, show it directly in a webbrowser, or save the chart PNG to disk.

Example 
>>> G = GChart('lc',['simpleisbetterthancomplexcomplexisbetterthancomplicated'])
>>> G.title('The Zen of Python','00cc00',36)
>>> G.color('00cc00')
>>> str(G)
'''http://chart.apis.google.com/chart?
    chd=s:simpleisbetterthancomplexcomplexisbetterthancomplicated
    &chco=00cc00
    &chts=00cc00,36
    &chs=300x150
    &cht=lc
    &chtt=The+Zen+of+Python'''
See testing for unit test and other examples
